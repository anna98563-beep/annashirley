Why we chose this topic?

Many students who have never worked with Visual Studio Code find it challenging to get started. The main difficulties typically include the large number of keywords, understanding their correct syntax and structure, and visualising how they want their website to ultimately look. We are no exception - however, we also want to use this project as an opportunity to demonstrate that building and styling a website can be an enjoyable and rewarding experience, as long as one grasps the underlying logic of how the tools work.

Project objectives:

  • to provide a brief introduction to HTML and what it is
  • to provide a brief introduction to CSS and what it is
  • to explain the key terms and keywords used in both languages
  • to demonstrate the practical use of tools for both HTML and CSS through the example of this website
