What is HTML?

HTML stands for HyperText Markup Language. It is not a programming language, it is a markup language, meaning it describes the structure and content of a webpage, telling the browser what each element is (a heading, a paragraph, an image, etc.).

A Brief History

HTML was created by Tim Berners-Lee , a British scientist working at CERN, in 1991. He needed a way to share documents between researchers over the internet. The first version was extremely simple - just 18 tags. Over the decades it evolved significantly. Today HTML5 is the universal standard used by every website in the world.

What is HTML Used For?

HTML is the skeleton of every webpage. Whenever you open a website, your browser reads HTML code and renders it visually. HTML defines:

  • headings and paragraphs
  • images and links
  • forms and buttons
  • the overall structure of the page

It always works together with CSS (visual styling) and JavaScript (interactivity).

Where Do You Write HTML?

You can write HTML in any text editor, but the most popular tool is Visual Studio Code (VS Code) — a free, lightweight editor by Microsoft.

Every HTML file is saved with the extension .html (e.g. index.html).

Most Common HTML Keywords (Tags)

HTML is built from tags, written in angle brackets like <tagname>. Most tags come in pairs: an opening tag and a closing tag.
